# 寶可夢指令相關

| 指令                      | 指令介紹                    |
| ----------------------- | ----------------------- |
| /endbattle              | 強制結束對戰，對戰時卡請稍後時可以使用     |
| /pokesell <遊戲ID> <編號位置> | 將身上隊伍指定位置的寶可夢發送給指定玩家    |
| /ivs <編號位置>             | 查看身上隊伍指定位置的的寶可夢IVS(個體值) |
| /evs <編號位置>             | 查看身上隊伍指定位置的的寶可夢EVS(努力值) |
| /acki                   | 查看身上隊伍的詳細資訊             |
| /wiki <圖鑑編號\|寶可夢英文名稱>   | 查看指定寶可夢圖鑑的資訊            |
| /ll                     | 查看過往的傳說寶可夢出現紀錄          |
| /giftignore <遊戲ID>      | 禁止玩家使用/pokesell指令給你寶可夢  |
